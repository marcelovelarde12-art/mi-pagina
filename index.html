¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje que se usa para dar estilo a las páginas web: colores, tamaños, márgenes, posiciones, y más. Permite separar el contenido (HTML) de la presentación visual.

Figuras con Colores

Podemos usar CSS para crear figuras simples como cuadrados, círculos y rectángulos solo usando colores y bordes.

Degradados en CSS

Los degradados permiten crear transiciones suaves entre colores. Se pueden hacer lineales o radiales usando background: linear-gradient().
